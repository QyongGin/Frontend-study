# 🧃 Product Card UI

상품 카드 형태의 UI를 HTML + CSS로 구현한 미니 프로젝트입니다.

## 📌 기능
- 상품 이미지, 이름, 설명, 가격, 구매 버튼
- 반응형 레이아웃 (모바일/태블릿/PC)
- 마우스 hover 애니메이션

## 📂 기술 스택
- HTML5
- CSS3 (Flexbox 또는 Grid)
